# Meme Wall

## Comment contribuer ?

1. **Fork** le projet.
2. **Ajoute** ton meme dans le dossier `memes/`.
   - ⚠️ Donne un nom unique à ton fichier (ex: `bug-java-theo.jpg` plutôt que `meme.jpg`).
3. **Commit** et **Push** sur ton fork.
4. **Ouvre une Pull Request** vers le repo principal.
5. Une fois la PR acceptée et mergée par un admin, le site se met à jour tout seul =
